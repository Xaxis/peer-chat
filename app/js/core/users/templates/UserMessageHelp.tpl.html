Welcome to peer-chat! The simple web based peer-to-peer chat client. For more information visit https://github.com/Xaxis/peer-chat.

/clear:
Clears content from a channels chat window. E.g. /clear

/help /h:
List application help information. E.g. /help

/ignore:
Ignore a specified user. E.g. /ignore anony9999

/join:
Used to connect to a given channel. E.g. /join #somechannel

/list /l:
Shows a list of available channels. E.g. /list
Shows a list of a peer's active channels. E.g. /list anony9999

/me:
Send an action message. E.g. /me Waves!

/msg /chat /notice /query:
Sends a message to a specific peer. E.g. /msg anony9999 Words for your face only!

/nick:
Change your user name. E.g. /nick myNewName

/part /close:
Leave this channel. E.g. /part

/partall /closeall:
Leave all channels. E.g. /partall

/ping:
Retrieve a time-to-response from peer. E.g. /ping PeerName

/whois:
Get information on a peer. E.g. /whois anony9999
